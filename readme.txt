千牛网是一个类似京东的电商平台
网址为http://localhost:8080/QianNiu
默认的欢迎页面是login.html
login.html是一个登录页面，本项目将更改为分布式架构，模块分为商品模块，用户模块
支付模块，网站静态模块，订单模块

商品模块分为食品模块，衣物模块，因为考虑到并发量可能会很大，所以食品和衣物分开写请求地址为*.do,
因为用户可以不用登录访问商品模块

用户模块分为 登录，注册，查看用户的购物车，添加购物车，生成订单

支付模块分为微信和支付宝支付

网站静态模块分为网站的静态页面和动态页面，网站的前端页面来源为    css库前端框架，和bootstrap前端框架
